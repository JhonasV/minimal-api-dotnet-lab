# minimal-api-dotnet-lab
A lab testing the dotnet 6 minimal api template

## Features
- Get One Todo
- Get All Todos
- Get One Post
- Get All Posts

## Technologies
- .NET 6 API Minimal (Minimal API Template)

## Installation

Restore the dependecies in the folder `minimal-api-dotnet-lab` root

```bash
dotnet restore
```


## Run the project
Go to the project root folder
```bash
dotnet run -p minimal-api-dotnet-lab
```

App must be running in ports:
```bash
 localhost:5001 OR localhost:5000
```
